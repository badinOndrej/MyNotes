1.  Download the PocketBook SDK for your firmware version and hardware level

    -   e.g. from [https://github.com/Sean-on-Git/PocketBook-SDK](https://github.com/Sean-on-Git/PocketBook-SDK)

2.  Check `sdkpath.txt`, modify the path in the file to point to the root of your copy of PocketBook SDK

3.  Build with

    `make`

    -   notes_app.app will be created upon success
